# ANTI-SLOP-KO — 한국어 / 한국 웹소설용 슬롭 사전

원본 `ANTI-SLOP.md` 는 영문 LLM 출력의 통계적 tell 을 다룬다.
이 파일은 **한국어 + 한국 웹소설** 환경에서 추가로 검출해야 할
슬롭 패턴을 정의한다. 두 사전은 함께 사용한다.

---

## Tier 1: Kill on sight (한국어 추상·정리어)

다음 단어·표현은 LLM 출력에서 통계적으로 과다. 작중 등장 시 즉시
재작성.

## Tier 2: 한국 웹소설 메타·먼치킨 어휘 (절대 금지)

이 작품은 mythic epic 톤이라 웹소설 메타 어휘를 절대 사용하지 않
는다.

- 스킬 / 스테이터스 / 능력치 / 레벨 / 레벨업
- 회귀 / 환생 / 전생 / 빙의 / 각성 / 각성자
- 사이다 / 고구마 / 먼치킨
- 시스템 / 메시지 / 알림창
- 아이템 / 인벤토리 / 장비
- 직업 / 클래스 / 직군 (게임 의미)
- 마나 / 포인트 / EXP / HP / MP
- 성좌 / 격 / 격이 다르다
- 차원 (게임 의미) / 다른 차원
- 계급장 / 등급 (S급·A급 같은 표기)

## Tier 3: 사극 클리셰 (강력 회피)

이 작품은 가상 사막 왕국이라 한국 사극 클리셰가 침투하면 톤이 깨진
다.

- "황공하옵니다 / 송구하옵니다 / 마마"
- "전하 / 폐하" 의 과잉 (신왕·신후 호칭은 별도 가상 명칭)
- "지엄하신 / 망극하옵신"
- "~이옵나이다 / ~하셨소이다"
- "감히 / 어인 일로"
- "충신·간신" 의 흑백 구도 어휘
- 한복 풍경 묘사 (사막 왕국이라 의상은 다름)

신왕·신후의 공식 발언에서도 절제. 한 문장에 1회 이내.

## Tier 4: 정서 직접 명명 (구체로 대체)

mythic epic 의 핵심 — 정서를 직접 말하지 않는다. 사물·행위·풍경으
로 보여준다.

회피:
- "그는 두려웠다 / 슬펐다 / 외로웠다 / 분노했다 / 화가 났다 / 허
  무했다 / 절망했다 / 안도했다"
- "그의 눈에 눈물이 맺혔다" 같은 직접 정서 묘사

대체 예시:
- "두려웠다" → "그는 천을 다시 묶었다. 두 번째 매듭에서 손이 멈
  췄다."
- "슬펐다" → "그는 노파의 약방 자리를 한 번 더 돌아보았다."
- "분노했다" → "그는 답을 하지 않았다. 답을 하지 않는 것이 답이었
  다."
- "외로웠다" → "광장의 12 동심원 중 어디에도 그의 자리는 없었다."

## Tier 5: 형용사·부사 폭주

한국어는 형용사·부사가 쉽게 누적된다. 슬롭 신호.

룰:
- **한 명사에 형용사 1개**. 두 개 = 한 개 자른다.
- **부사 -게 / -이 / -히 형태가 한 단락에 3개 이상 = 재작성**.
- **"매우 / 너무 / 정말 / 굉장히 / 엄청나게 / 훨씬"** 같은 강조
  부사는 회당 2-3회 이내.
- 색깔·재질·소리 묘사는 1회 1단어 — "검고 무겁고 차가운 모래" → "검은
  모래" 또는 "차가운 모래" 중 하나만.

## Tier 6: 메타 코멘트 (화자의 정리어)

회피:
- "그렇게 그는 ~"
- "이 모든 것은 ~"
- "결국 그는 알게 됐다"
- "그것이 시작이었다" (한 작품에 1-2회 이내)
- "이때만 해도 그는 몰랐다" — 한국 웹소설에서 가장 흔한 슬롭. 절대
  금지.
- "운명이 그를 ~" — 신화적이지 않고 멜로드라마. 금지.

## Tier 7: 한국 웹소설 회차 호흡 슬롭

회차 호흡 자체가 만드는 슬롭 패턴.

### 회피 패턴

1. **회차 첫머리에 직전 회차 사건 즉답**: 회차마다 직전의 cliffhanger
   를 즉시 푸는 것은 호흡을 무너뜨린다. 다른 시점·다른 장면으로 짧
   게 환기 후 진입.
2. **회차 중간의 정보 덤프**: 세계관·과거 설명을 한 단락에 3문장 이
   상 = 회차 호흡 깨짐. 정보는 행위·대사로 분배.
3. **회차 마지막의 텍스트 cliffhanger**: 텍스트만으로 "그러나 그는
   곧 알게 될 것이었다" 같은 closing — 슬롭 톱급. 시각적 이미지로
   대체.
4. **이미 본 것을 다시 설명**: 회차 시작 시 직전 회차 요약 금지.
   독자는 정주행 중이다.
5. **반복 수사·반복 표현 누적**: 한 인물의 같은 표현이 회차마다
   반복되면 = 캐릭터 강조가 아니라 슬롭. 회당 1회까지.

### Tier 7-5 보강: voice 시그니처어 회당 임계 (cycle 7 실측)

이 작품의 voice 는 색·아우라·결·잔향·호흡 어휘가 정체성이다. 따라서
일반 반복 수사 (회당 1회 룰) 와 별개로 다음 시그니처어는 별도 임계를
적용. cycle 7 voice 시필 4편 (회 1·7·9·50, 평균 4,100자) 실측 기반.
220화 정주행 누적 피로 통제용.

| 시그니처어 | 회당 cap | cycle 7 실측 (회 1/7/9/50) | 주의 |
|---|---|---|---|
| 한 호흡 | 12 | 3 / 15 / 8 / 10 | 회 7 15회 = 임계 초과. 회당 12 이내. |
| 잔향 | 10 | 0 / 16 / 5 / 8 | 회 7 16회 = 임계 초과. 잊힌 신 power 회차도 cap 10. |
| 한 단계 (더) | 5 | 2 / 4 / 0 / 6 | 회 50 6회 = 임계 초과. 색의 변화 묘사 회차도 cap 5. |
| 결 (색·아우라) | 30 | 21 / 30 / 19 / 57 | 회 50 57회 = 임계 2배 초과. 색 묘사 핵심 어휘여도 cap 30. |
| 안개 | 10 | 14 / 0 / 5 / 1 | 회 1 14회 = 임계 초과. 의식 장면도 cap 10. |
| 답하지 않는 것이 답이었다 | 1 | 0 / 2 / 0 / 0 | 회 7 2회 = 임계 초과. 회차당 max 1, 작품 전체 5회 이하. |
| 답하지 않았다 (변형) | 2 | — | 변형도 누적 합산. |
| 한 사람의 / 한 명의 / 한 호흡의 | 5 | — | "한 X 의 Y" 패턴 누적 cap. |

**갱신 룰**:
- 작성 후 grep 로 실측. cap 초과 시 재작성.
- 220화 누적 통제: 매 50화마다 시그니처어 누적 빈도 측정 → 누적 그래프
  가 한 어휘에 집중되면 어휘 풀 갱신.
- voice 시그니처어 빈도가 cap 안인데도 "단조롭다" 신호가 오면 어휘
  풀 자체를 다양화 (예: "한 호흡" → "한 박자", "한 결", "한 자리" 등
  의 변형 도입).

### 좋은 패턴

- 회당 진전 1-2개 + 감정 전환 1개 + cliffhanger 1개
- 회차 마지막 = 한 컷의 시각적 이미지 (웹툰화 의식)
- 회차 첫머리 = 시점 인물 명시 + 전 회와는 다른 호흡으로 진입
- 한 회 안에서 호흡 변화 = 사색 1단락 → 사건 3단락 → 대사 2단락 →
  사색 1단락 → cliffhanger.

## Tier 8: 캐릭터별 voice 일관성 위반

각 인물의 voice 가 회차마다 흔들리면 슬롭. 캐릭터별 통계 룰:

- 하셈: 한 대사가 평균 2-5단어. 한 회 안에서 평균 단어 수가 6 단어
  넘으면 재검토.
- 시샤: 욕설·외성 구어. 정중한 단어("그대" 등) 사용 시 = 재작성.
- 라이샤르: "그대" 호명, 한자어. 외성 구어 사용 시 = 재작성.
- 무한: 약초 비유, 시적·느림. 빠른 외성 구어 사용 시 = 재작성.
- 잊힌 신: 사람의 목소리가 아닌 음. 일상 어휘 사용 시 = 재작성.

## Tier 9: 사건 인플레이션 (mythic epic 의 호흡 보존)

웹소설 톤은 사건을 빠르게 넣으려는 압력이 있다. mythic 톤은 사건
의 무게가 중요하다. 균형:

- **한 회당 인물 죽음 1명 이상 = 재검토**. 죽음의 무게를 보존.
- **한 회 안에 새 인물 3명 이상 등장 = 재검토**. 인물 도입 호흡.
- **새 장소 도입 시 묘사 1단락 이상 + sensory hook 1개 이상**.
  도입 압축 = 톤 깨짐.
- **잊힌 신의 속삭임 = 회당 1-2회까지**. 너무 자주 = 신비 약화.

## 검진 체크리스트 (회차 작성 후)

회차 한 편 쓴 직후 다음을 자가 점검:
1. Tier 2 (먼치킨 어휘) 한 단어라도 들어갔나? → 즉시 제거.
2. Tier 1 (추상·정리어) 회당 2개 이상? → 재작성.
3. Tier 4 (정서 직접 명명) 한 회 1개 이상? → 행위로 대체.
4. 회차 마지막 = 시각적 이미지인가? → 아니면 재작성.
5. 회차 첫머리 = 직전 cliffhanger 즉답인가? → 환기 추가.
6. 한 인물의 voice 가 다른 회차와 일관성 있는가?
7. 회당 진전 + 감정 전환 + cliffhanger 모두 들어갔나?
8. 분량 5,000-5,200자 ±200자 이내인가?
